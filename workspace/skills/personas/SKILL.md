20 built-in personas for specialized help.

## Usage

- Activate: "Use Dev", "Switch to Chef Marco", "Activate Dr. Med"
- List: "List all personas" or `/persona list`
- Exit: "Exit persona mode" or `/persona exit`

## CLI

```bash
python3 scripts/persona.py --list          # List all
python3 scripts/persona.py --activate luna  # Activate
python3 scripts/persona.py --current        # Current
python3 scripts/persona.py --reset          # Reset
```
