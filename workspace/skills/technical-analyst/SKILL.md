# Technical Analyst

Analyze weekly price chart images for trends, levels, and probabilistic scenarios.

## Core Principles

1. **Pure chart analysis** — only observable chart data, no news/fundamentals
2. **Systematic approach** — structured methodology per chart
3. **Probabilistic scenarios** — weighted future outcomes

## Workflow

### Per chart image:

1. **Trend** — direction (up/down/sideways), strength, duration
2. **Support/Resistance** — horizontal levels, trendlines, confluences
3. **Moving Averages** — price vs 20/50/200-week MAs, alignment, crossovers
4. **Volume** — trend, spikes, confirmation/divergence
5. **Patterns** — reversals, continuations, candlestick formations
6. **Scenarios** — 2-4 probability-weighted outcomes

### Scenario Structure

Each scenario includes:
- **Name**: e.g., "Bull Case: Breakout Above Resistance"
- **Probability**: must sum to 100%
- **Supporting factors**: 2-3 minimum
- **Target levels** and **invalidation level**

### Typical Framework

| Scenario | Probability |
|---|---|
| Base Case | 40-60% |
| Bull Case | 20-40% |
| Bear Case | 20-40% |
| Alternative | 5-15% |

## Output

Save as `[SYMBOL]_technical_analysis_[YYYY-MM-DD].md`

## Quality Standards

- No subjective language ("I think", "I feel")
- Specific price levels, not vague descriptions
- Both bullish and bearish possibilities presented
- Invalidation levels for every scenario
