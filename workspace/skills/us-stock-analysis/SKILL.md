# US Stock Analysis

Structured frameworks for analyzing US stocks.

## Analysis Types

1. **Basic Info** — quick overview with key metrics
2. **Fundamental** — financials, valuation, business quality
3. **Technical** — indicators, trends, patterns
4. **Comprehensive Report** — all approaches combined
5. **Peer Comparison** — side-by-side stock evaluation

## Data Sources

Use web search with ticker + specific data needed:
- Yahoo Finance, MarketWatch, Seeking Alpha, Bloomberg
- SEC filings (10-K, 10-Q) for detailed financials
- TradingView for technical data

## Fundamental Workflow

1. Gather 3-5 year revenue, earnings, cash flow trends
2. Analyze profitability (margins, ROE, ROIC)
3. Assess balance sheet (debt, cash, liquidity)
4. Perform valuation (P/E, PEG, P/B, EV/EBITDA vs peers and historical)
5. Identify risks and red flags

## Technical Workflow

1. Identify trend direction and strength
2. Locate support/resistance levels
3. Analyze RSI, MACD, Bollinger Bands, volume
4. Identify chart patterns
5. Generate outlook with risk/reward

## Report Structure

- Executive summary with recommendation
- Investment thesis (bull + bear cases)
- Fundamental analysis section
- Technical analysis section
- Valuation with fair value range
- Risk assessment + catalysts
